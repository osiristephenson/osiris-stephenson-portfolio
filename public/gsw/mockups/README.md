# iPhone 17 Pro Max Mockups (Renders)

Drop your iPhone 17 Pro Max render images into this folder using **these exact filenames**:

- `hero-center.png` — front/straight-on render (best quality)
- `hero-left.png` — left-tilt/isometric render
- `hero-right.png` — right-tilt/isometric render

Optional (used for feature + gallery phones if you want the same device look everywhere):

- `device-front.png` — clean front render for repeated use

## Notes

- PNG is preferred (JPG works too).
- Keep the background on the renders (black/gradient) — the page will layer your GSW+ screenshots on top of the render.
- After adding files, refresh the page (hard refresh: Cmd+Shift+R).
